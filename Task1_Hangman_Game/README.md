# Task 1 - Hangman Game

## Description
This is a simple text-based Hangman Game developed using Python.

## Features
- Random word selection
- 5 predefined words
- Maximum 6 wrong guesses
- Input validation
- Beginner-friendly code
- Console-based gameplay

## Concepts Used
- Python
- Random Module
- While Loop
- If-Else Statements
- Lists
- Strings

## How to Run

Clone the repository

```bash
git clone <your-repository-link>
```

Go inside the folder

```bash
cd codealpha_tasks/Task1_Hangman_Game
```

Run

```bash
python hangman.py
```
